cyber~Wednesday Day 0

Welcome to the new bi-weekly report format from cyber!

As most of you know, previously we used to report weekly in our TG channel in an open format. But nothing lasts forever and its time to move to a new format. An asynchronous format that will:

  • Eliminate the need for a monolithic team leader
  • Make the report more interactive to the public
  • Include more information from each full-time member of the team
  • Be more efficient and creative than before

Last week, we outlined a new set of guidelines for the reports. Each report will be a combined work from the team. It will be pushed to our GitHub and thereafter published for AMA (ask me anything) on the #fuckgoogle TG channel. For now, questions from the community will be answered openly in the channel.
